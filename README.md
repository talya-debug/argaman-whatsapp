# בוט וואטסאפ - ארגמן

בוט שמאזין לקבוצת וואטסאפ, יוצר משימות ב-Firestore ושולח תזכורות.

## התקנה

```bash
npm install
```

## הפעלה

```bash
node src/index.js
```

## התחברות ראשונה

1. הפעל את השרת - יופיע קוד QR בטרמינל
2. פתח וואטסאפ בטלפון → הגדרות → מכשירים מקושרים → קשר מכשיר
3. סרוק את הקוד
4. ברגע שמופיע "מחובר לוואטסאפ" - הבוט פעיל

## מציאת מזהה קבוצה (GROUP_ID)

1. הפעל את הבוט בלי GROUP_ID (השדה ריק)
2. שלח הודעה בקבוצה הרצויה
3. הבוט ידפיס בטרמינל: `GROUP_ID לא מוגדר. מזהה קבוצה זו: XXXXX@g.us`
4. העתק את המזהה לקובץ `src/config.js` בשדה GROUP_ID
5. הפעל מחדש

## פריסה ב-Render.com (חינם)

1. העלה את הקוד ל-GitHub (בלי תיקיית auth_info)
2. היכנס ל-render.com וצור Background Worker חדש
3. חבר את הריפו מ-GitHub
4. הגדר:
   - Build Command: `npm install`
   - Start Command: `node src/index.js`
5. בפעם הראשונה תצטרך להתחבר דרך הלוגים של Render (לסרוק QR)

**שים לב:** בתוכנית החינמית של Render השרת נכבה אחרי 15 דקות ללא פעילות.
לפתרון קבוע - שקול שרת VPS (כמו DigitalOcean) או הרצה על מחשב ביתי.

## מה הבוט עושה

- **יצירת משימות** - כל הודעה בקבוצה הופכת למשימה ב-Firestore
- **תזכורת יומן עבודה** - כל יום א-ה בשעה 18:00
- **תזכורת משימות** - כל יום א-ה בשעה 09:00 (משימות שתאריך היעד שלהן היום)
- **סיכום שבועי** - כל יום ראשון בשעה 08:00
